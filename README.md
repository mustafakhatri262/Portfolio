
  # Personal Portfolio Website

  A modern personal portfolio website built with React.

  ## Features

  - Responsive design
  - About Me section
  - Skills showcase
  - Projects section
  - Contact section
  - Smooth animations

  ## Tech Stack

  - React
  - Tailwind CSS
  - JavaScript
  - Vite

  ## Getting Started

  ```bash
  npm install
  npm run dev
  ```
